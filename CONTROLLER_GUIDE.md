# 🎮 手把控制指南 Controller Guide
**RobotControl2026 — FRC 2026 賽季**

> 本機器人使用 **Xbox 手把** 進行遙控操作，支援場地導向（Field-Oriented）與機器人導向（Robot-Centric）兩種驅動模式。

## 按鍵功能總表

| 按鍵 | 觸發方式 | 功能 | 說明 |
|------|----------|------|------|
| **左搖桿 Y 軸** | 持續 | 機器人前後移動 | 場地導向：相對於場地前後 |
| **左搖桿 X 軸** | 持續 | 機器人左右移動 | 場地導向：相對於場地左右 |
| **右搖桿 X 軸** | 持續 | 機器人旋轉 | 順/逆時針轉向 |
| **左肩鍵 (LB)** | 按住 | 🎯 自動瞄準並射擊 | AutoAimAndShoot：自動對準 Hub + 依距離計算 RPS + 準確時自動出球 |
| **右扳機 (RT > 10%)** | 按住 | 🎯 自動瞄準並射擊 | 功能同左肩鍵（含 Shuffleboard 監控頁籤） |
| **左扳機 (LT > 10%)** | 按住 | 🟡 吸球 (Intake) | IntakeRoller 正轉吸球 + up_to_shoot 反轉擋球，防止球衝入射手 |
| **A 鍵** | 按住 | 🏷️ 自動對準 AprilTag | Drive2Tag：自動行駛到最近 AprilTag |
| **B 鍵** | 按住 | ⏪ 輸送帶反轉 | 反向運行 Transport，吐球（解除卡球） |
| **X 鍵** | 按住 | ▶️ 輸送帶正轉 | 手動啟動 Transport 送球 |
| **Menu 鍵 (Button 8)** | 單按 | 🔄 重置 IMU | 緊急重置陀螺儀方向（紅方→180°，藍方→0°） |
| **右搖桿按下 (RS)** | 單按 | 🔀 切換駕駛模式 | 切換場地導向 ↔ 機器人導向 |

## 功能詳細說明

### 🎯 自動瞄準並射擊（AutoAimAndShoot）
**觸發**：左肩鍵 或 右扳機（> 10%）

執行流程：
1. **射手待命**：射手整場以 55 RPS 待命（不需要手動啟動）
2. **自動旋轉**：機器人旋轉至面向 Hub 目標（由位姿估算計算角度）
3. **慣性補償**：移動中射擊時，根據底盤速度預測未來位置，提前調整瞄準角度
4. **自動計算轉速**：根據到 Hub 的距離，套用二次曲線公式計算最佳 RPS：
   $$RPS = 1.165 \times d^2 + 1.573 \times d + 35.216$$
5. **等待就緒**：射手達到目標轉速後自動觸發輸送帶出球
6. **持續射擊**：持續按住按鍵期間保持射擊狀態

> ⚠️ 距離範圍：1.0m – 5.0m（超出範圍自動夾限至 35–75 RPS）

---

### 🟡 吸球（Intake Roller）
**觸發**：左扳機（> 10%）

- **IntakeRoller 正轉**：滾輪以目標轉速吸球
- **Transport 正轉**：輸送帶往機器人內部輸送
- **up_to_shoot 反轉**：上膛馬達反轉**擋住球**，防止球直接衝入射手飛輪
- 放開後三者同時停止

> 💡 吸球時 up_to_shoot 反轉是刻意設計，確保球停在輸送帶中段待射。
> 需要推球進射手時，按 **X 鍵** 手動正轉 Transport。

---

### 🏷️ 自動對準 AprilTag（Drive2Tag）
**觸發**：A 鍵（按住）

- 機器人自動行駛至對準最近 AprilTag（偏移量：X = -1.15m，Y = 0.0m）
- 射手繼續維持 idle 旋轉（不停機）
- 鬆開按鍵後恢復手動控制

---

### ⏪ 輸送帶反轉（Reverse Transport）
**觸發**：B 鍵（按住）

- 反向運行輸送帶，將球從射手側退回
- 用途：卡球排除、重新定位

---

### ▶️ 輸送帶正轉（Run Transport）
**觸發**：X 鍵（按住）

- 手動啟動輸送帶，強制送球到射手
- 通常搭配手動射手使用（測試用）

---

### 🔄 重置 IMU（Reset IMU）
**觸發**：Menu 鍵（單按）

- 根據目前**聯盟顏色**自動設定正確方向：
  - 藍方（Blue Alliance）→ 0°（面向 Hub）
  - 紅方（Red Alliance）→ 180°（面向 Hub）
- 場地導向依賴 IMU，比賽前或方向錯誤時使用

---

### 🔀 切換駕駛模式
**觸發**：右搖桿按下（單按）

| 模式 | 說明 |
|------|------|
| **場地導向 Field-Oriented（預設）** | 推左搖桿向場地「前方」就往場地前方走，不受機器人朝向影響 |
| **機器人導向 Robot-Centric** | 推左搖桿向「前」就往機器人面對的方向走 |

## 開賽前確認清單

- [ ] 確認 Limelight 看得到 Hub（`AutoAimAndShoot` 需要）
- [ ] 按 **Menu 鍵** 確認 IMU 方向已重置
- [ ] 確認聯盟顏色（Driver Station 正確設定）
- [ ] 測試吸球功能（左扳機，確認 up_to_shoot 反轉擋球）
- [ ] 測試射擊功能（左肩鍵，靜止狀態瞄準）
- [ ] 確認射手已在旋轉（Shuffleboard Shooter 分頁的 Current RPS 應 ≈ 55）

---

## 自主期開始

`teleopInit()` 執行時機器人會自動：
1. 從 Limelight MegaTag2 讀取位置並**重置 Odometry**
2. 手把震動 **雙次提示音** 確認就緒
